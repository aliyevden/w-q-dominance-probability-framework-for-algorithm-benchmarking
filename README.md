Files in this depository are to accompany the w--q: A Dominance-Probability Framework for Evaluating Optimization Algorithms paper by Aliyev D. A. (2026)
1) all_tour_lengths.parquet file contains the tour length distributions distributions for 36 methods on 70 TSP datasets obtained using the R code from
    https://github.com/aliyevden/TPN-matrices-for-TSP-heuristics.
   Note: The illustrative examples in the paper use a subset of scores corresponding to CI, tp_a_CI, tp_s_CI, tpn_a_CI, and tpn_s_CI methods.
2)  The R code for the Illustrative Examples is available upon request and will be made public soon.
